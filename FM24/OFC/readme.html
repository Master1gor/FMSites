Football Manager rule files for OFC v24.1

In this package, you will find rule files for OFC Member Nations.

I did my best to use realistic formats for all the leagues that are included. The rule files contain:

  • real divisions and cups with real formats from 2023
  • teams from the 2023 season
  • all nations have transfer windows, registration periods, tv dates, match rules, prize money - when I was able to find this in my online research

Compatibility

  • FM Database 24.0.0

Installation Guide

Copy the .fmf files for the nations you want into the folder "<My Documents>\Sports Interactive\Football Manager 2024\editor data\". The folder can be different if you have changed it in the FM24 preferences. The leagues should then appear the next time you start a new game in FM24 and chose advanced setup.

Changelog

  • OFC / OFC_ALL : refactored qualifications to the OFC Champions League

  • New Zealand: National League no longer runs as parent competition, this enables a better view of all subleagues
  • New Zealand: better handling of Wellington Phoenix Reserves
  • New Zealand: removed foreigner rule for Wellington Phoenix Reserves (because main team is in Australia, all players were otherwise listed as foreigners)
