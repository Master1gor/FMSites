Football Manager rule files for OFC v24.4

In this package, you will find rule files for OFC Member Nations.

I did my best to use realistic formats for all the leagues that are included. The rule files contain:

  • real divisions and cups with real formats from 2023
  • teams from the 2023 season
  • all nations have transfer windows, registration periods, tv dates, match rules, prize money - when I was able to find this in my online research

Compatibility

  • FM Database 24.3.0 (only)

Installation Guide

Copy the .fmf files for the nations you want into the folder "<My Documents>\Sports Interactive\Football Manager 2024\editor data\". The folder can be different if you have changed it in the FM24 preferences. The leagues should then appear the next time you start a new game in FM24 and chose advanced setup. Make sure you have the checkbox "add players to playable teams" checked. This is recommended, otherwise you may end up with a lot of greyed-out players.

Coefficients

I added coefficients for the OFC Champions League. Unfortunately, the coefficients cannot be viewed in the game. I was unable to activate that page, which is there for e.g. Europe. So, Because it is invisible it is totally intransparent what the rankings for the countries are.
The coefficienta are averages based on the number of teams in continental competitions (e.g. if New Zealand has 2 teams in the competition then the points gained by those teams is divided by 2) Basically, nations get points for each team from that country:

  • 2p for each match win
  • 1p for each match draw (a game that goes to penalties is concidered a draw)
Things I added:
  • 3p for all teams that start in the Qualifying Round (5p if you use the file which also adds Tuvalu and Kiribati)
  • 4p for 1st place in Group Stage
  • 2p for 2nd place in Group Stage
  • 1p for reaching the Final
