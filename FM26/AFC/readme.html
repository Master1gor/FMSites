Football Manager rule files for AFC v26.4

In this package, you will find rule files for AFC Member Nations.

I did my best to use realistic formats for all the leagues that are included. The rule files contain:

  • real divisions and cups with real formats from 2025/26
  • teams from the 2025/26 season
  • all nations have transfer windows, registration periods, tv dates, match rules, prize money - when I was able to find this in my online research

Compatibility

  • FM Database 26.0.0 (only)

License

Football Manager rule files for AFC v26.4 by MasterIgor is licensed under Attribution-NonCommercial-NoDerivatives 4.0 International

The rule files are free, but every tip will greatly improve my motivation to provide updates and other federations, because I spent a great deal of free time researching and editing. So if you really enjoy the rule files, then every small contribution is greatly appreciated. Support Me.

How to install

Copy the .fmf files for the nations you want into "Editor Data" folder. This folder can be different depending on where you got the game or what system you have. I am assuming you know where this folder is. If not, check other sources on where you can locate it for your setup.

Please remove any older files so that you do not have multiple versions for the same country.

Also, it is better to have the checkbox "add players to playable teams" checked. Otherwise you may end up with a lot of greyed-out non-database players.

Testing

all rules files have been tested by me in the editor and in the actual game (unemployed mode)
of course, it is possible that there are issues and mistakes in the rule files. If you find an issue, you can contact me by E-Mail.

AFC Continental Competitions v26.2
[Cup] AFC Champions League Elite
[Cup] AFC Champions League Two
[Cup] AFC Challenge League
[Cup] South East Asian Club Championship (ADDED)

  • I have recreated the AFC Club Competitions because the in-game versions had multiple issues. Here is a list of what my version (should) fix:
    • Preliminary Round of ACLE for more than 4 teams fixed according to real rules (in-game version had losers advance to next round instead of winners)
    • East and West teams are in correct groups (in-game version had them mixed)
    • ACLE League Phase teams from same nation do not play eachother (in-game version did not care about this)
    • Round of 16 in ACLE now in correct format (in-game version did not seperate east and west)
    • KO Round in ACL2 now in correct format (in-game version did not seperate east and west)
    • KO Round in ACGL now in correct format (in-game version did not seperate east and west)
    • ACLE Quarter final to final now played in host city (in-game version chose different neutral stadium for each match)
    • Final of ACL2 and ACGL alternates between home stadium of East and West teams every year (in-game version just picked home stadium randomly)
    • Foreigner rules removed and real rules about home-grown players added
    • coefficients fixed (in-game version gave round of 16 teams twice the points, and no points for final participants)
    • Qualified teams from Nepal now picked from A-League (in-game version picked from Super League)
    • maybe more, but I don't remember...
  • WARNING: Since this file replaces all the in-game AFC competitions, there is a chance that it may not work well together with other editor files or other in-game competitions (though I must say, the in-game version is a poor implementation). Please be aware of the risk. I have designed the rules so they work with my rule files for each country and I recommend to use that setup. If you wish to use this file together with files from other editors, please do some tests before starting the save to observe the behaviour.
  • NOTE: there is a helper competition called "AFC Continental Competitions". This is not a real competition, but only contains stages to seperate east and west and setup the qualified teams for ACLE, ACL2 and ACGL. (I left it visible so that I could analyze it in case of issues).
  • v26.2 Changed and hopefully fixed registration issues.
